# sjzxd-tools
# 使用说明

三角洲小程序数据比较多，看不同的数据要点击不同的功能，对于我来说有点繁琐，而且不能实时统计数据，所以编写了该脚本，将所有数据都汇聚到一起并进行处理和整合。

## 1.前提说明

建议使用 **Windows Terminal** 作为运行脚本的终端，因为在脚本中加了颜色和图标，使用其他的终端可能会造成乱码。

现在搜索中找输入**商店**

### 2.2 微信扫码登录

```

```

暂时只支持微信登录，QQ登录问题暂时未解决
